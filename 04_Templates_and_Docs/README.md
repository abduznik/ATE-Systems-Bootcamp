# 04 - תבניות ותיעוד

תיקייה זו מכילה את התבניות הנדרשות להגשת המשימות. שימוש בתבניות אלו הוא חובה כדי להבטיח אחידות ומקצועיות.

## קבצים בתיקייה
- `BOM_Template.csv`: רשימת רכיבים לייצור.
- `PR_Template.md`: תבנית להגשת Pull Request (יש להעתיק את התוכן לתיאור ה-PR ב-GitHub).
- `Pinout_Diagram_Blank.drawio`: קובץ בסיס לתיעוד חיבורי הפינים של המערכת.
