# 05 - אתגרי לוגיקה (Logic Challenges)

השלב הסופי ב-Bootcamp בוחן את היכולת שלכם לכתוב קוד יעיל, מהיר ונקי.

## חוקי הברזל
1. **No-Delay:** שימוש ב-`delay()` יפסול את המשימה באופן אוטומטי.
2. **Registers over DigitalRead:** במשימות ביצועים, העדיפו שימוש ב-Bit Masking על פני פונקציות Arduino סטנדרטיות.

## האתגרים
- **State Machine:** מימוש מכונת מצבים לניהול רצף בדיקה (Idle -> Testing -> Pass/Fail).
- **Debounce:** מימוש מנגנון Debounce ללחצן ללא השהיית המעבד.
- **Bit Manipulation:** שליטה על מספר יציאות במקביל באמצעות Register Control.
