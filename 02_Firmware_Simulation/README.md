# 02 - סימולציית Firmware

לפני הניגש לחומרה פיזית, עלינו לוודא שהלוגיקה הבסיסית עובדת.

## המשימה
עליכם להריץ קוד בסיסי המבצע Blink ללד המובנה ובמקביל שולח נתונים ל-Serial Monitor בתדר של 1Hz.

## דרישות טכניות
- שימוש ב-`millis()` בלבד לצורך תזמון. **שימוש ב-`delay()` אסור בהחלט.**
- הגדרת Baud Rate של 115200.
- הקוד חייב להיות מתועד (Comments) באנגלית.
