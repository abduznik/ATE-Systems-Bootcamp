# 02 - סימולציית Firmware

לפני הניגש לחומרה פיזית, עלינו לוודא שהלוגיקה הבסיסית עובדת.

## המשימה
עליכם להריץ קוד בסיסי המבצע Blink ללד המובנה ובמקביל שולח נתונים ל-Serial Monitor בתדר של 1Hz.

## דרישות טכניות
- שימוש ב-`millis()` בלבד לצורך תזמון. **שימוש ב-`delay()` אסור בהחלט.**
- הגדרת Baud Rate של 115200.
- הקוד חייב להיות מתועד (Comments) באנגלית.

## איך בודקים? (How to Test)

התהליך הבא מגדיר את סביבת העבודה ואת אופן אימות המשימה:
1. **כתיבת הקוד:** השתמשו ב-**Arduino IDE 2.x** כדי לכתוב ולדבג את הקוד שלכם. ודאו שאין שגיאות קומפילציה.
2. **הרצת הסימולציה:** היכנסו לאתר [Wokwi.com](https://wokwi.com), צרו פרויקט ESP32 חדש וטענו לתוכו את הקבצים `Blink_and_Serial.ino` ו-`diagram.json`.
3. **וולידציה (Validation):** 
   - ודאו שהלד הכחול (או המובנה) מהבהב בקצב קבוע.
   - פתחו את ה-Serial Monitor בסימולציה וודאו שזמן המערכת מודפס בכל שנייה בדיוק, ללא השהיות.
