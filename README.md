# ATE Systems Bootcamp - Industrial Sandbox 2026

ברוכים הבאים לסביבת ה-Sandbox של ה-Bootcamp. פרויקט זה נועד להכשיר הנדסאי אלקטרוניקה ותוכנה לעבודה בסטנדרטים הגבוהים ביותר של תעשיית ה-ATE (Automated Test Equipment).

## קונספט ה-Sandbox
הפרויקט מדמה סביבת עבודה אמיתית במרכז פיתוח. אין כאן "תרגילים", אלא משימות הנדסיות (Tasks) הדורשות תכנון, מימוש, תיעוד ובקרת איכות. כל שינוי בקוד או בחומרה חייב לעבור דרך Pull Request (PR) וסקירה טכנית.

## Toolchain - כלי עבודה
כדי לעמוד בסטנדרט המקצועי, עליכם להתקין ולהשתמש בכלים הבאים בלבד:
- **גרסאות תוכנה:**
  - **GitHub Desktop:** לניהול גרסאות וסנכרון מול השרת.
  - **Arduino IDE 2.x:** לפיתוח Firmware.
  - **KiCad 10.0:** לתכנון עריכת מעגלים מודפסים (PCB). **חובה להשתמש בגרסה 10.0 ומעלה.**
  - **Draw.io Desktop:** ליצירת דיאגרמות בלוקים ו-Flowcharts.
- **דרייברים נדרשים:**
  - CP210x USB to UART Bridge
  - CH340/CH341 Serial Driver

## מתודולוגיית עבודה - 7-Day Sprint
הפרויקט מתנהל במתכונת של ספרינט בן 7 ימים.
1. **יום 1-2:** הקמת סביבה, Git, וסימולציה ראשונית.
2. **יום 3-5:** תכנון חומרה ב-KiCad וכתיבת לוגיקה לא חוסמת (Non-blocking).
3. **יום 6:** אינטגרציה, תיעוד סופי וייצוא PDF.
4. **יום 7:** הגשת PR סופי. 

**אזהרה:** אי-עמידה בלוחות הזמנים או כישלון במיזוג (Merge) של ה-PR עד סוף היום השביעי יביא לסיום הפרויקט באופן מיידי.

## מבנה המערכת
המערכת מורכבת מ-5 שלבים מרכזיים, המפורטים בתיקיות הפרויקט. יש לעקוב אחר ה-README בכל תיקייה לפי הסדר הכרונולוגי.
